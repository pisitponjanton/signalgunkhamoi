โครงงาน Signal Gunkhamoi เป็นระบบอุปกรณ์ป้องกันการโจรกรรมอัจฉริยะ (Smart Anti-Theft System) ที่พัฒนาขึ้นโดยใช้ Arduino UNO R4 WiFi ทำหน้าที่ตรวจจับการเคลื่อนไหวของวัตถุหรือบุคคลผ่าน Ultrasonic Sensor (HC-SR04) หากมีการตรวจพบสิ่งเคลื่อนไหว ระบบจะส่งเสียงเตือนผ่าน Buzzer และส่งข้อมูลแจ้งเตือนผ่านเครือข่าย Wi-Fi ไปยัง API Server ซึ่งพัฒนาโดยใช้ Next.js / Node.js เพื่อจัดเก็บข้อมูลลงฐานข้อมูลและแจ้งเตือนไปยังผู้ใช้ผ่านทางอีเมล

อุปกรณ์สามารถเข้าสู่โหมดการตั้งค่า Wi-Fi ได้ผ่านระบบ Access Point Mode เพื่อรับค่าการเชื่อมต่อและ Device Code ซึ่งจะถูกบันทึกลงใน EEPROM สำหรับการทำงานในโหมดปกติ ระบบจะใช้ข้อมูลที่บันทึกไว้ เชื่อมต่อกับเครือข่ายอัตโนมัติและทำงานตรวจจับทันทีเมื่อเปิดเครื่อง โดยมี LED แสดงสถานะการทำงานของระบบแบบเรียลไทม์ และสามารถรีเซ็ตค่าได้ด้วยปุ่มกด (Push Button)

ฝั่งเซิร์ฟเวอร์ทำหน้าที่รับข้อมูลการแจ้งเตือนจากอุปกรณ์ผ่านโปรโตคอล HTTPS โดยมีฐานข้อมูล MySQL / SQLite จัดเก็บเหตุการณ์ทั้งหมด และระบบ Nodemailer ใช้สำหรับส่งอีเมลแจ้งเตือนไปยังผู้ใช้เมื่อมีการตรวจพบเหตุบุกรุก ข้อมูลทั้งหมดสามารถดูได้แบบเรียลไทม์ผ่านเว็บไซต์หลักของระบบที่พัฒนาโดย Next.js

โดยสรุป ระบบนี้เป็นต้นแบบของเทคโนโลยี IoT เพื่อความปลอดภัยที่สามารถเชื่อมต่ออินเทอร์เน็ต ส่งข้อมูลแบบอัตโนมัติ และแจ้งเตือนเหตุการณ์ได้แบบเรียลไทม์ เหมาะสำหรับการประยุกต์ใช้ในพื้นที่ส่วนบุคคล เช่น เรือ บ้าน หรือสถานที่ที่ต้องการการเฝ้าระวังอัตโนมัติ
